Materi

Eksponen merupakan suatu simbol atau angka yang menunjukkan seberapa banyak suatu angka yang dikalikan.

Dari situ, kita bisa lihat kalau eksponen merupakan perkalian berulang sebanyak pangkatnya. Dalam hal ini, seberapa banyak simbol atau angka dikalikan, disebut sebagai pangkat. Sementara itu, angka yang dikalikan disebut sebagai basis.

Contoh soal : Bentuk 34 dapat dinyatakan sebagai …. jawab : Ingat bahwa bentuk an= a x a x a x … x a (sebanyak n kali). Hal ini membuat 34 = 3 x 3 x 3 x 3.

Contoh Soal Eksponen Kelas 10

1. Akar-akar persaman eksponen (32 pangkat x) – (12 pangkat 3x) +1 + 91 = 0 adalah x1 dan x2. Jika x1 > x2, maka nilai x1 – x2 =….

Penyelesaian:

Misal (3 pangkat x) = p maka persamaannya adalah sebagai berikut. (p pangkat 2) – 30p + 91 = 0 (p – 3) (p – 27) P = 3 P = 27 3x = 3 pangkat 1 3x = 3 pangkat 3 X = 1 X = 3 X1 – X2 3 – 1 = 2
